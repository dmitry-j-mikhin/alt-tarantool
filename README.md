# alt-tarantool

This repository is used to build tarantool 2.10.3 for Alt P10.

Assembly steps:
```Shell
./build.sh
```
As a result, an `out` folder with build artifacts is created.
Compiled rpm packages are also located here - [https://github.com/dmitry-j-mikhin/alt-tarntool/releases/tag/v2.10.3](https://github.com/dmitry-j-mikhin/alt-tarantool/releases/tag/v2.10.3).
